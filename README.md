# REG-knowledge-sharing
A public facing repository for knowledge sharing activities at the Research Engineering Group, such as REG Tech Talks.
See the [wiki](https://github.com/alan-turing-institute/REG-knowledge-sharing/wiki) for the schedule and location.
There is also a Slack channel: #dss.

## Tuesday Tech Talks

The REG team holds tech talks on a variety of topics, usually on Tuesday lunchtimes at 13:00. See [here](https://github.com/alan-turing-institute/REG-knowledge-sharing/wiki/Lunchtime-Tech-Talks) for the latest schedule.

Anyone at Turing is welcome to present at the Tech Talks.
